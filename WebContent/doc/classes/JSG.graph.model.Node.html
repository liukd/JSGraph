JSG.graph.model.Node Class

Extends JSG.graph.model.GraphItem
Module: JSG.graph.model

The main usage of a node is to represent a JSG.graph.model.GraphItem which can be connected to other nodes. The connection between nodes is established by attaching an JSG.graph.model.Edge to a JSG.graph.model.Port of each node. So every node manages a list of ports to which an edge can connect.
The default shape of a node is a JSG.graph.model.shapes.RectangleShape.
