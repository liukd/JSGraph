JSG.ui.viewer.ScrollableViewer Class

Extends JSG.ui.viewer.GraphViewer
Module: JSG.ui.viewer

A JSG.ui.viewer.GraphViewer subclass which uses a JSG.ui.scrollview.ViewPanel to show its content. This view panel is embedded within a scroll panel which is either passed to the constructor function or is an instance of JSG.ui.ScrollPanel by default.
To detect mouse events which occur on JSG.ui.scrollview.ScrollBar of a ScrollPanel all mouse handling methods of current default JSG.graph.interaction.Interaction are decorated by JSG.ui.viewer.ScrollableViewerInteractionDecorator.
layout should be called if the size of the enclosing container or element has changed and setZoom can be called to zoom displayed content.
