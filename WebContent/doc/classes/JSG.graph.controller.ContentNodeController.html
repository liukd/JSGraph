JSG.graph.controller.ContentNodeController Class

Extends JSG.graph.controller.NodeController
Module: JSG.graph.controller

This controller handles the behavior of JSG.graph.model.ContentNodes.
Since a ContentNode has a so called ContentPane sub item, this controller adds a corresponding JSG.graph.controller.ContentPaneController to its child list. This ContentPaneController can be retrieved via getContentPaneController.

Returns the sub controller for specified location or undefined if none could be found.
The controller look up can be affected by the flags and condition parameter. The condition must be a function which gets called with any possible controller as parameter. If passed controller is valid, the function should return true.

    The location to look at.

    One of the predefined flags to affect to controller look up.

  • [condition] Function optional
