JSG.ui.graphics.GraphicSystem Class

Module: JSG.ui.graphics

The GraphicSystem is the main class to handle events and drawing requests.
Therefore a reference to an html canvas element is required to which the GraphicSystem will register itself as listener to the various key, touch and mouse events. These events are transformed from native events to their corresponding API JSG.ui.events.Events and then propagated to a registered JSG.graph.interaction.InteractionHandler or, if no InteractionHandler was set, to the main root JSG.ui.View.
A drawing request can be triggered by a call to paint. However, it is rarely necessary to call paint directly. In most cases a repaint is required as a result of handling an event notification. Then the preferred way is to simpy set the doRepaint property to true.

Finally, if the GraphicSystem is no longer needed destroy should be called to perform a clean up and to deregister all registered listeners.
