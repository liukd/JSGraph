JSG.graph.attr.FormatAttributes Class

Extends JSG.graph.attr.AttributeList
Module: JSG.graph.attr

This AttributeList defines default attributes for formatting a JSG.graph.model.GraphItem. The members can be used to directly influence the format of a GraphItem including fill and line as well as shadow formatting.

Constructor

JSG.graph.attr.FormatAttributes

(
  • [mapExpr]
)

Parameters:

  • [mapExpr] JSG.graph.expr.MapExpression optional

    An optional MapExpression which contains predefined format attributes.

Example:

 var node = new JSG.graph.model.Node();
         // get format attributes for node
         var format = node.getFormat();
         format.setFillColor("#FF0000");
         format.setLineColor("#00FF00");

Parameters:

  • attributeList JSG.graph.attr.AttributeList

    An AttributeList containing the Attributes to retain.

    An optional condition Function.

Returns:

Boolean:

var retainFunc = function(attr1, attr2) {
    return (attr1 != undefined) && (attr2 != undefined) && attr1.getValue() == attr2.getValue();
}

setParent

(
  • list
)

Inherited from JSG.graph.attr.AttributeList:

Sets a new parent for this AttributeList.
Note: to support store and restore parent correctly it is required that it is attached to a JSG.graph.model.GraphItem within the same JSG.graph.model.Graph as the GraphItem of this list belongs to. Otherwise if the parent is not attached to a GraphItem it should be an instance of JSG.graph.attr.Template.
Finally note that it is allowed to pass undefined in order to remove any formerly applied parent.

Parameters:

  • list JSG.graph.attr.AttributeList

    The new parent list to use for this AttributeList. Specifying undefined is allowed.
