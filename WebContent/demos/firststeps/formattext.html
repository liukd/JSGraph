Formatting Text Nodes in JS Graph

Overview

This chapter explains in more detail, how text formats can be assigned to GraphItems. A TextNode is always a GraphItem on its own that is a sub item of another GraphItem. This way a label (TextNode) can be attached to Nodes like rectangle or ellipsis or to lines and edges. Text formats can be applied to TextNodes being attached to a node or to a line, but also to the parent GraphItem.

By default the TextNode uses the text format being defined with the parent GraphItem. This way the formatting is inherited from the parent. If you change the parent text format, all contained labels or TextNodes will use this format. If you need an individual text format for one or all of the TextNodes within a GraphItem you can assign individual formats to each or some of the TextNodes.

In addition to the text format of a GraphItem you can insert (limited) HTML code as text into a TextNode. The HTML tags will be used to format and structure the content of the TextNode. The set of allowed tags is currently limited to the Paragraph, List tags and Spans. In addition formatting tags for bold, italic are valid. The text alignment, font size, color and family can be assigned using styles.

Text Formatting

Text can be formatted using font settings. You can apply a font name, font size and the font styles. In addition you can set the text position within the parent node. You can have more than one label attached to a node. Each label can have its own font settings. In the following sample you can see how the font settings are applied and their effect.
